# 🏡 Airbnb Clone – Backend

A backend system that powers an Airbnb-like rental platform. It supports features such as user authentication, property listings, bookings, payments, reviews, and admin management.

---

## ✨ Key Features

- User registration and login for guests and hosts
- Property listing and management by hosts
- Booking system with availability checks
- Secure payment processing via third-party gateway
- Review system for guests and hosts
- Admin dashboard for managing users and listings
